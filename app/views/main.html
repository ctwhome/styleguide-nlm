Technologies used in Front-end

We use gulp as the task runner. To install gulp is necesary to have:

  1. Install Node.js

  2. To run gulpfile from the FrontEnd path in the project run "npm install". (Only necesary when chekout the project for first time, or updates of the gulpfile)

Go to the documentation section to see the description of the gulp tasks
